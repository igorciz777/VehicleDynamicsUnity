# Vehicle Dynamics Unity Project

## Overview
This project aims to simulate realistic vehicle dynamics within the Unity game engine and it's PhysX physics engine. It provides a foundation for a vehicle simulation model, where the unsprung mass is seperately modeled from the sprung mass, meaning that the suspension is physically represented.

## Features
- Realistic suspension modeling
- Custom wheel collision model
- Pacejka tire model implementation
- Highly adjustable parameters for vehicle tuning
- Non-linear engine throttle model
- Basic input handling for vehicle control, including Unity-DirectInput package for FFB wheel support
- Plot class to visualize vehicle data in real-time
